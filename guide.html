            # 产品功能介绍

            # 数据提取

            连接不同位置数据源，采集到不同格式和内容的物流信息数据。数据采集模块负责制定采集规则并采集数据。数据源为mysql数据库、hdfs存储、minio存储。数据文件格式分别为txt、csv、xls等。可动态管理（创建，修改，删除）数据源。

            # 数据存储

            将采集数据按规则治理后存入DM8数据库中，并提供质量数据及补录功能。

            # 数据治理

            动态设定治理规则，数据治理模块负责制定治理规则，并按规则治理数据。可动态配置治理规则，如数据聚合，数据去重，异常值检测，数据转换等。包含以下治理规则：（1）“身份证号治理”，客户身份证必须符合规范。（2）物流单规则，物流信息里客户必须在客户数据中。（3）治理完成数据和异常数据分开存放，异常数据提供处理界面，并能重新治理。

            # 数据可视化

            对外提供数据分析服务。根据“数据治理”结果，生成“物流信息分析系统”，系统数据全部来源数据存储共享模块，包含如下功能模块：（1）港口的吞吐量分析（2）港口不同类型货物吞吐趋势（3）港口货物吞吐同比环比（4）不同货物吞吐占比（5）不同货物流向分析（6）不同类型货物堆场流转周期分析
